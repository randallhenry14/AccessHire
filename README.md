# AccessHire

AccessHire is an ADA-compliant hiring platform demo built as a static HTML/CSS/JS app.

## Preview
This repository is configured for GitHub Pages. Once deployed, the app can be opened from the repo's Pages link or at:

https://randallhenry14.github.io/AccessHire/

## Run locally
1. Open `index.html` in your browser, or
2. Serve the folder with a local server such as `npx serve .`

## Deploy
Push to `main` and GitHub Actions will publish the site automatically.
